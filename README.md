# 시간이 멈춘 고대 도시

## 프로젝트 소개
시간 조작을 핵심 메카닉으로 하는 횡스크롤 2D 퍼즐 액션 게임

플레이어 주변에 시간이 흐르는 영역(시간 필드)을 생성해 퍼즐을 해결하는 게임입니다.

## 개발 환경
- Unity 
- C#
- Git

## 스테이지
| 스테이지 | 설명 |
|------|------|
| 1스테이지 | 시간이 멈춘 고대 유적 - 시간 필드로 시간을 흐르게 함 |
| 2스테이지 | 생생한 고대 도시 - 시간 필드로 시간을 멈추게 함 |
| 보스 스테이지 | 과거 시점 진입, 능력 반전 |
